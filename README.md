#  Full Stack Portfolio App

A full-stack portfolio web application built with React and Node.js.  
It includes authentication, project management, notes system, and dark mode.

---

## ✨ Features

-  User authentication (Login / Register)
-  Dashboard with user access
-  Notes system (Create, Edit, Delete)
-  Projects showcase
-  Add projects dynamically
-  Tech stack tags for projects
-  Dark mode toggle
-  Responsive UI
-  Animated cards and hover effects

---

## 🛠 Tech Stack

### Frontend
- React
- React Router
- CSS3

### Backend
- Node.js
- Express
- JWT Authentication

### Database
- SQLite / PostgreSQL (depending on setup)
